# Kamaynikasyon - Filipino Sign Language Learning Application

## Overview

Kamaynikasyon is a mobile application designed to help users learn Filipino Sign Language (FSL). This project serves as a capstone project for undergraduate thesis requirements, demonstrating practical application of software development skills in creating an educational tool for accessibility and language learning.

## Features

- Interactive sign language lessons
- Dictionary of FSL signs
- Progress tracking for learners
- User-friendly interface for beginners

## Tech Stack

- Android (Kotlin/Java)
- Firebase for backend services
- Admin panel built with React/Vite

## Installation

1. Clone the repository.
2. Open in Android Studio.
3. Build and run on an Android device or emulator.
